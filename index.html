Berenice Morales's Portfolio

Welcome to my data analytics and data science portfolio! Here, I showcase a collection of projects that demonstrate my expertise to derive insights from data. Whether working individually or collaboratively in teams, each project highlights my proficiency in languages and tools such as SQL, Python, AWS, Streamlit, and Tableau.

Alberta Shelter Capacity Website

Deployment of a website using the AWS Platform and Python Streamlit library to provide information on emergency shelters in Alberta.

Airbnb Listings Analysis

Utilized Python to analyze Airbnb listings for investors. The analysis included creating clear visuals to identify pricing patterns, occupancy rates, and critical factors affecting property investment decisions.

U.S. Energy Consumption Analysis

Utilized Python and SQL to analyze the Trends and Shifts in U.S. Energy Consumption.

Drugs and Substance Abuse Analysis

Utilized Python to analyze the factors that influence drug consumption behavior using statistical methods like decision trees, gradient boosting, logistic regression, and random forest analysis.

Customer Behaviour Analysis

Utilized Python to segment customers based on Recency, Frequency, and Monetary (RFM) scores and applied K-means clustering to refine customer segmentation. I analyzed transactional data to perform a market basket analysis.

An Overview of People with Disabilites in Mexico

Utilized Python and Tableau to examine the population with disabilities in Mexico, analyzing distribution, activity engagement, and educational accessibility to identify challenges and patterns within the demographic.
